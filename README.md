# Leetcode-Solutions

A clean and organized repository containing solutions to various algorithmic problems from LeetCode, implemented using **Java**. This project tracks core data structures and algorithm paradigms learned during interview preparation.

---

## 📂 Repository Structure

The code is organized categorically into folders based on problem difficulty:

```text
├── Easy/
│   └── [Problem-Name]/
│       └── Solution.java
├── Medium/
│   └── [Problem-Name]/
│       └── Solution.java
└── Hard/
    └── [Problem-Name]/
        └── Solution.java
```

---

## 🚀Running Solutions Locally

You can compile and run any solution file directly from your terminal using standard Java development tools.

### Step 1: Navigate to the specific problem directory
```bash
cd Easy/[Problem-Name]/
```

### Step 2: Compile the Java class
```bash
javac Solution.java
```

### Step 3: Run the bytecode
```bash
java Solution
```

*Note: To feed custom parameters into the solution class locally, you can temporarily append a `public static void main(String[] args)` method inside the class.*

---

## 🧠 Topics & Frameworks
* **Data Structures:** Arrays, Hash Tables, Linked Lists, Binary Trees, Graphs, Heaps, and Stacks.
* **Algorithms:** Two Pointers, Sliding Window, Binary Search, Dynamic Programming, Backtracking, and Greedy choices.
* **Java Collections Framework:** Built entirely using native Java classes like `HashMap`, `ArrayList`, `HashSet`, and `PriorityQueue` without external dependencies.
